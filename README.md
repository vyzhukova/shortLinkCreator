# Short Link Creator

Сервис для сокращения ссылок с возможностью создания кастомных алиасов, статистикой переходов, регистрацией пользователей и автоматическим удалением неиспользуемых ссылок.

## Технологии
- Python 3.11, FastAPI
- PostgreSQL, SQLAlchemy
- Redis (кэширование)
- JWT-аутентификация
- Docker, docker-compose

## Запуск
иметь у себя докер десктоп. Запустить команду
```bash
docker-compose up --build
```
## Описание запросов
1. Регистрация нового пользователя  
POST /register
2. Вход (получение токена)  
POST /login
3. Создание короткой ссылки    
POST /links/shorten
4. Редирект по короткой ссылке  
GET /{code}
5. Получение статистики по ссылке  
GET /links/{code}/stats
6. Удаление ссылки (требуется аутентификация)  
DELETE /links/{code}
7. Обновление оригинального URL (требуется аутентификация)  
PUT /links/{code}
8. Поиск ссылок по оригинальному URL  
GET /links/search?original_url={url}
9. Группировка ссылок по проектам (требуется аутентификация)  
GET /links/by_project
10. Список истекших ссылок  
GET /links/expired
